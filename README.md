# StayNestVilla

StayNestVilla is a .NET solution that includes an ASP.NET Core Web API and an ASP.NET Core MVC UI application. The API provides RESTful endpoints for managing villas (rental properties), and the MVC UI consumes the API using HttpClient to deliver a real-world client experience.

## Features
- ASP.NET Core Web API with CRUD endpoints for villas
- ASP.NET Core MVC UI (Razor Views) that consumes the API via HttpClient
- DTO-based request/response models
- Entity Framework Core with SQL Server (Code-First + Migrations)
- Repository Pattern and Dependency Injection
- Authentication and Authorization (JWT / Identity)
- OpenAPI documentation (Swagger / Scalar)

## Tech Stack
- .NET (ASP.NET Core Web API, ASP.NET Core MVC)
- C#
- Entity Framework Core
- SQL Server
- JWT / ASP.NET Core Identity
- Swagger / Scalar
- HttpClient
